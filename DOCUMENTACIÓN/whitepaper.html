Technical Whitepaper

Antigravity Quantum is an institutional-grade algorithmic trading system designed for high-frequency low-latency execution across Crypto (Binance) and Equity (Alpaca) markets. This document outlines the technical specification, internal logic, and security protocols of the bot.

1. System Architecture

The system is built on an asynchronous event-driven architecture using Python 3.11 and asyncio. It creates a non-blocking loop that handles websocket data streams, signal processing, and order execution concurrently.

[ Binance WS ] <----> [ Data Normalizer ] ----> [ Quantum Engine ]
|
[ Alpaca API ] <----> [ AI Analyst ] --------> [ Signal Bus ] ----> [ Execution Manager ]

Core Components

  • Bot Async (`bot_async.py`): The central entry point. Manages the aiogram polling loop and the background QuantumEngine task.
  • Trading Manager (`AsyncTrainingSession`): Handles exchange interactions, position sizing, and order lifecycle.
  • Dynamic Cooldown: Prevents signal spam/over-trading by tracking volatility and ATR per asset.

2. Quantum Engine

The Quantum Engine is the decision-making brain. It processes raw market data (OHLCV) into actionable signals.

Signal Generation Process

  1. Data Ingestion: Fetches up to 1000 candles for analysis.
  2. Indicator Calculation: Computes RSI, Bollinger Bands, ATR, MACD, and ADX using pandas_ta.
  3. Strategy Evaluation: Passes data through active Strategy Classes (Trend, Scalping, etc.).
  4. Consensus Filter: Checks signals against global market sentiment (AI) and Risk constraints.

3. Trading Strategies

The bot employs specific logic classes found in strategies/classes/.

Trend Following

CONFIDENCE > 0.75

Identifies sustained momentum. Enters when price breaks Bollinger Bands with high ADX. Uses an EMA200 filter for directionality.

Mean Reversion

CONFIDENCE > 0.80

Contrarian strategy. Buys when RSI < 30 and price hits lower Bollinger Band; Shorts when RSI> 70 and price hits upper band.

Scalping (High Frequency)

CONFIDENCE > 0.65

Exploits micro-movements on 1m/5m timeframes. Tight Stop Losses and quick Take Profits.

4. Risk Management Protocols

Security is the primary directive. The system cannot be overridden to bypass these checks.

ü¶à Shark Sentinel

A background daemon that monitors Bitcoin's global health. If BTC drops more than 3% in 1 hour (Flash Crash), the bot:

  • Cancels all open BUY orders.
  • Freezes new Long positions.
  • Sends a "Black Swan" alert to all admins.

Circuit Breaker

If the Daily PnL drops below a configurable threshold (e.g., -5%), trading is suspended for 24 hours to prevent "revenge trading" spirals.

5. AI Sentiment Layer

The system queries OpenAI (GPT-4) to analyze textual market data.

Usage: /analyze [SYMBOL]

The AI reviews technical indicators + "simulated" news events to provide a verbal summary and a "Sentiment Score" (-1.0 to 1.0). If the Sentiment Score is negative, Long Technical signals are invalidated.
